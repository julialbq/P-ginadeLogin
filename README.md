# Página de Login

## Idealização

- Página de login responsiva com modal de "esqueci minha senha". Entrega com o objetivo de praticar estilização com componentização do CSS, organização de pastas, responsividade e desenvolvimento de modal

## Tecnologias

- HTML
- CSS (mobile first)
- Javascript
